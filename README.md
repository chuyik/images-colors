Image Colors
========

获取图片主颜色，用法参见 `test/`。

# 特性
- 支持获取 JPG 和 PNG 图片的主颜色列表，并按占比倒序返回
- 支持网络图片和本地图片

# 下载
JNPM 项目地址：http://npm.m.jd.com/package/@o2team/images-colors
   
```bash
npm install @o2team/images-colors --registry=http://registry.m.jd.com
```
